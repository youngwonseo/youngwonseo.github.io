Dataflow: Qwik Start - Template

클라우드 빅쿼리 데이터셋과 테이블 생성

텍시라이버 데이터텟 생성

bq mk taxirides

빅쿼리 테이블 초기화

bq mk \ –time_partitioning_field timestamp \ –schema ride_id:string,point_idx:integer,latitude:float,longitude:float,\ timestamp:timestamp,meter_reading:float,meter_increment:float,ride_status:string,\ passenger_count:integer -t taxirides.realtime

스토리지 버킷 생성

export BUCKET_NAME= gsutil mb gs://$BUCKET_NAME/

클라우드 빅쿼리 데이터셋과 테이블 GCP콘솔에서 사용하기

  1. 왼쪽 메뉴의 [BigQuery]클릭
  2. 프로젝트 클릭
  3. CREATE DATASET 클릭
  4. taxirides 추가

생성한 DATASET클릭후 CREATE TABLE

왼쪽 메뉴의 Storage Browser에서 [CREATE BUCKET]

Bucket 이름은 글러벌에서 유니크 해야한다.

Run the Pipeline

Navigation menu에서 Dataflow [Create job from template] [Job name]
