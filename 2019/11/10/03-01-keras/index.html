신경망 시작하기#1

[케라스창시자에게 배우는 딥러닝]의 3장 내용을 정리한 것입니다.

1. 신경망의 구조

신경망은 다음과 같은 요소로 구성되어 있습니다.

1.1 네트워크(모델)을 구성하는 층(layer)

층은 하나 이상의 텐서를 입력으로 받고 하나 이상의 텐서를 출력합니다. 또한 입/출력과 더불어 가중치(weight)를 상태를 가지는데 입/출력에 대해 가중치를 학습합니다(경사하강법을 통해 학습). 풀고자 하는 문제(데이터에 따라)에 따라 신경망은 적절한 층을 가져야 합니다(각 층에 대한 자세한 설명은 앞으로의 포스트를 참조해주세요).

  • 완전 연결 층(fully connected layer)
  • 밀집 연결 층(densely connected layer)
  • 순환층 (reccurent layer)
  • 합성곱 층(convolution layer)

아래는 가중치가 없는, 즉 학습이 이루어지지 않는 층입니다(가중치는 없지만 각각 필요한 용도가 존재합니다.).

  • flatten layer
  • pooling layer
  • dropout layer

케라스에서는 호환 가능한 층들을 엮어 데이터 변환 파이프라인을 구성함으로써 딥러닝 모델을 만듭니다.

from keras import models
from keras import layers

model = models.Sequential()
model.add(layers.Dense(32, input_shape=(784,)))
model.add(layers.Dense(10))

model에 Dense층을 쌓아 올려 층을 만듭니다. 첫번째 층에는 인풋텐서가 784의 크기를 가져야된다는 것을 알지만 두번째 층은 명시 되어 있지 않습니다. 이것은 앞층의 출력(32개)이 다음층의 입력으로 지정되기 때문입니다. 위 모델의 시각화하면 다음과 같습니다.

1.2 모델: 층이 모여 구성된 네트워크

딥러닝은 바로위에 명시된 층(layer)이 모인 비순환 유향 그래프(Directed Acyclic Graph, DAG)입니다. 가장 일반적인 예는 하나의 출력이 하나의 아웃풋을 만들어내는 모델입니다.

층이 쌓여 모델, 즉 네트워크(자료구조의 그래프 형태를 뜀, 그래서 DAG) 구조를 만드는데 이는 가설 공간(hypothesis space)을 정의합니다. 입력 데이터를 출력 데이터(학습데이터는 입/출력데이터로 이루어져 있습니다)로 매핑하는 딥러닝에서는 이러한 모델에 따라 가설 공간(가능성 있는 공간)이 정해지는데 모델이 잘못정해지면 입력 데이터를 출력 데이터로 매핑(매핑은 입력데이터를 출력데이터로 변환하는 텐서연산을 의미합니다)할 수 없습니다. 그러니 입출력 데이터에 적합한 모델을 선택하는 것이 정말 중요합니다!

딥러닝에서 딥러닝, 모델, 네트워크(네트워크 구조)는 모델을 지칭하는 용어입니다.

1.2 입력데이터와 타겟(target)

1.3 손실함수(loss function)

손실함수는 비용함수(cost function)나 목적함수(object function)라고도 불리는 요소로 모델을 오차률(오차률을 계산한다는 것은 정확도를 계산한다는 것과 같습니다)로 표현이 가능합니다. 즉 모델을 학습할때는 손실함수가 최소화되도록 학습합니다.

1.4 옵티마이져(optimizer)

손실함수를 기반으로 모델이 어떻게 학습(여기서 학습은 가중치의 업데이트를 의미합니다)할지 결정합니다. 앞에서 학습은 경사하강법에 의해 진행된다고 설명했는데 옵티마이져는 이 경사하강법이 어떻게 가중치를 업데이트 시킬지 결정합니다. 즉 가중치를 업데이트하는 방법이 여러가지가 존재한다는 의미이죠.
