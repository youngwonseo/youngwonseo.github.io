자바 - String Class

스트링객체는 자바에서 가장 많이 사용되는 객체입니다. 잘 사용하면 상관없지만 잘못 사용하면 메모리에 많은 영향을 줍니다.

스트링을 잘못 사용한 사례

자바의 스트링은 불변객체입니다. 즉 다음과 같은 연산을 진행할때 메모리상으로는 어떤동작이 발생하는지 생각해 봅시다.

String str = "";
str += "hi ";
str += "My";
str += " ";
str += "name";
str += " ";
str += "is";
str += " ";
str += "youngwon";

StringBuilder와 StringBuffer

StringBuilder와 StringBuffer의 차이점을 아는것도 자바의 스트링을 이해하는데 있어서 중요한 요소입니다.

StringBuffer는 쓰레드에 안전(ThreadSafe)합니다.
