자바스크립트 Array 객체

05 Oct 2018    ·   

Array

자바스크립트에는 많은 표준 내장 객체들이 존재합니다. 표준 내장 객체만 제대로 파악하고 있어도 멀리 돌아가는 일(삽질..)을 줄이고 좀 더 표준화(standardization)된 코드를 생산 할 수 있습니다. 오늘은 표준 내장 객체중 Array에 대해서 살펴보겠습니다. 모든 예제는 ES6기준의 코드입니다. 이전 버전의 자바스크립트를 사용한다면 모질라 공식사이트(https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array) 의 폴리필부분을 살펴보시기 바랍니다.

Methods

Array.prototype.concat()

두개 또는 그이상의 배열을 합쳐 새로운 배열을 반환합니다. 이때 원본의 배열들은 아무런 영향을 받지 않습니다.

var array1 = [1,2]
var array2 = [3,4]
var array4 = [5,6]

var array3 = array1.concat(array2)
var array5 = array1.concat(array2,array3,array4)
var array6 = array1.concat(1,[2,3])

Array.prototype.copyWithin()

[].copyWithin()

Array.prototype.entries()

배열을 대상으로 key/value쌍의 반복자를 반환합니다.

var array = [1,2,3]
var iterator = array.entries()

for(let e of iterator) {
  console.log(e)
}

Array.prototype.every()

콜백에 대해 배열의 모든 요소가 만족하는지 검사합니다.

function isBelowThreshold(currentValue, index, array) { //index, array생략가능
  return currentValue < 40;
}
var array1 = [1, 30, 39, 29, 10, 13];
console.log(array1.every(isBelowThreshold));

Array.prototype.fill()

정해진 값으로 배열을 채웁니다. 정확하게는 첫 번째 인자값으로 두번째 인자에 대한 인덱스 부터 세번째 인자에 대한 인덱스 까지 채웁니다.

[1,2,3].fill(4) //[4, 4, 4]
[1,2,3].fill(1, 2) //[1, 2, 2]
[1,2,3].fill(4, 0, 2) //[4, 4, 3]

Array.prototype.find()

콜백에 대해 만족하는 첫번째 배열값을 반환합니다. 비슷한 함수로 findIndex가 존재하며 값이 아닌 인덱스를 반환합니다.

var array1 = [5, 12, 8, 130, 44];
var found = array1.find(function(element, index, array) { //index, array생략가능
  return element > 10;
});
console.log(found);
// expected output: 12

Array.prototype.flat()

주어진 인자(depth)만큼의 배열을 하나의 배열로 만듭니다. depth의 default값은 1입니다.

var arr1 = [1, 2, [3, 4]];
arr1.flat(); 
// [1, 2, 3, 4]

var arr2 = [1, 2, [3, 4, [5, 6]]];
arr2.flat();
// [1, 2, 3, 4, [5, 6]]

var arr3 = [1, 2, [3, 4, [5, 6]]];
arr3.flat(2);
// [1, 2, 3, 4, 5, 6]

var arr4 = [1, 2, , 4, 5]; //빈배열 제거
arr4.flat();
// [1, 2, 4, 5]

Array.prototype.shift()

배열의 첫번째 요소를 제거후 반환합니다.

var array = [1,2,3,4,5,6]
console.log(array.shift()) //1
console.log(array) // [2,3,4,5,6]

Array.prototype.slice()

배열을 잘라 sub array를 반환합니다. 이때 자를 부분의 처음 인덱스와 마지막 인덱스를 인자로 넘겨줍니다.

var animals = ['ant', 'bison', 'camel', 'duck', 'elephant'];

console.log(animals.slice(2));  //마지막 인덱스 생략시 배열의 마지막 까지
// expected output: Array ["camel", "duck", "elephant"]

console.log(animals.slice(2, 4));
// expected output: Array ["camel", "duck"]

console.log(animals.slice(1, 5));
// expected output: Array ["bison", "camel", "duck", "elephant"]

Array.prototype.sort()

배열의 요소들을 정렬합니다.

Array.prototype.splice()

배열의 특정 인덱스에 있는 요소를 삭제하거나 특정 인덱스에 추가합니다. 인자는 start, deleteCount , item…으로 deleteCount가 0이면 삽입, 이외의 숫자이면 삭제를 합니다.

var company = ['airbnb', 'apple', 'microsoft', 'google', 'deepmind']
//두 번째에 새로운 값 삽입
company.splice(2,0,'amazon') //['airbnb', 'apple', 'amazon', 'microsoft', 'google', 'deepmind']

//세번째 부터 한개 삭제
company.splice(3,1) //['airbnb', 'apple', 'amazon', 'google', 'deepmind']

Array.prototype.toString()

배열을 문자열로 반환합니다.

var monthNames = ['Jan', 'Feb', 'Mar', 'Apr'];
var myVar = monthNames.toString(); // 'Jan,Feb,Mar,Apr'을 myVar에 할당.

Array.prototype.unshift()

새로운 값의 배열의 맨 처음에 입력합니다. 참고로 Array.prototype.push는 새로운 값은 배열의 마지막에 추가합니다. 삭제된 값은 반환됩니다.

var arr = [1, 2];

arr.unshift(0); // result of call is 3, the new array length
// arr is [0, 1, 2]

arr.unshift(-2, -1); // = 5
// arr is [-2, -1, 0, 1, 2]

arr.unshift([-3]);
// arr is [[-3], -2, -1, 0, 1, 2]
