자바스크립트 데이터 타입

자바스크립트는 var 하나로 모든 데이터를 처리한다. 이것 때문에 몇몇 개발자들이 자바스크립트에 하나의 원시형 밖에 존재하지 않는다고 오해할 수 있는데 자바스크립트의 원시형(primitive)타입에는 다음과 같이 5개의 종류가 존재한다(ES5부터 Symbol이 추가).

  • 문자열(String)
  • 숫자(Number)
  • 불리언(Boolean)
  • null
  • undefined

또 한 다음과 같은 래퍼(Wrapper)가 존재한다.

  • String
  • Boolean
  • Number

다음 코드에서, 단순 문자열이 length라는 프로퍼티를 가질 수 있는 이유는 자바스크립트 엔진이 length의 호출을 보고 래퍼클래스로 변환해 주기 때문이다.

	var str = "hello world";
	console.log(str.length); //String(str).length, 결과 : 11

length를 호출하기 위해 생성한 String 객체는 저행이 마치면 마로 GC의 대상이 된다.
