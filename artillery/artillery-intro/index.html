소프트웨어 부하 테스트 도구 Artillery 소개

2021-02-17 / 서영원

여기서는 소프트웨어 부하 테스트 도구인 Artillery를 소개합니다.

Artillery를 설치하기위해서 다음과 같은 노드 패키지를 설치합니다(노드가 설치되어있지 않다면 노드를 먼저 설치해주세요).

npm install -g artillery

테스트 항목 정의

Artillery를 사용해서 테스트를 수행하기 위해서 테스트 시나리오를 yaml파일로 정의합니다.

cpu-test.yaml

config:
  target: "http://localhost:8080"
  phases:
    - duration: 60
      arrivalRate: 5
      name: Warm up
scenarios:
  - name: "CPU Test"
    flow:
      - get:
          url: "/cpu-test"

설명을 간략하게 하면 다음과 같습니다.

  • target : localhost:8080가 테스트를 수행할 서버
  • duration : 60초 동안 테스트를 수행
  • arrivalRate : 매초 5명의 가상유저를 생성

cpu-test.yaml이 정의된 디렉토리에서 다음과 같은 명령어를 실행합니다. 아래의 명령어는 테스트 결과를 report.json으로 출력합니다.

artillery run --output report.json ./cpu-test.yaml

테스트의 결과로 생성된 report.json을 다음의 명령어로 시각화로 리포팅된 결과를 확인할 수 있습니다.

artillery report ./report.json
