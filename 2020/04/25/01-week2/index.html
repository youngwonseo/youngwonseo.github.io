April 25, 2020 • Youngwon Seo

Evaluating models

2주차 Key Concepts

  • Calculate true positives, true negatives, false positives, false negatives.
  • Calculate sensitivity and specificity
  • Calculate Positive Predictive Value (PPV) and Negative Predictive Value (NPV).
  • Understand confidence intervals, ROC curve, and F1 score.

의료 분야에서 진단에 대한 결정은 높은 영향력을 가집니다. 그래서 진단에 대한 딥러닝 모델을 개발시 모델에 대한 평가..

  1. 정확도 메트릭의 단점, sencitivity, specificity 정화도로 부터 의료 평가
  2. 의사들을 의사결정을 위한 예측 값

다음은 정확도를 조건부 확률로 해석하는 과정입니다. +는 positive, 즉 질병(disease)으로 결정, -는 negative 질병이 아닌(normal)으로 결정을 의미합니다.

  • Sensitivity : 질병일 경우 질병으로 예측할 확률, P(+ disease), (true positive rate)
  • Specificity : 질병이 아닐경우 질병이 아닌것으로 예측할 확률, P(- normal), (true negative rate)

Sensitivity와 Specificity를 사용해 정확도를 다시 표현해 봅니다.

  • Prevalence : 질병이 발병할 학률, P(disease)

질병이 아닌 확률은 P(1-prevalence), P(1 - P(disease))입니다.

이것은 정확도를 sensitivity와 specificity의 가중치 평균으로 ..

예제를 살펴보겠습니다.

  • Sensitivity : P(+ disease) = #(+ and disease) / #(disease) = 2/3 = 0.67
  • Specificity : P(- normal) = #(- and normal) / #(normal) = 6/7 = 0.86
  • Prevalence : P(disease) = #(disease) / #(total) = 3/10 = 0.3
  • Accuracy : Sensitivity X prevalence + Specificity X (1 - prevalence) = 0.67 x 0.3 + 0.86 x 0.7 = 0.8
PPV(positive predictive value) : P(disease +)
NPV(nagative predictive value) : P(normal -)

Sensitivity, Specificity와 PPV, NPV의 관계를 Confusion matrix를 통해 알아니다.
