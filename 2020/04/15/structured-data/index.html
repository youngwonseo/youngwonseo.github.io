텐서플로우 - 구조화된 데이터

April 15, 2020 • Youngwon Seo

본 글은 텐서플로우 공식 사이트의 구조화된 데이터를 정리한 글입니다.

  • 특성 열(feature_columns)

텐서플로우에서 제공하는 특성 열 feature_columns의 종류

numeric column bucketized column categorical column embedding column

categorical_column_with_hash_bucket feature cross

수치형일 경우 실제 데이터와는 독립적으로 feature의 이름(column의 이름)으로만으로 구성할 수 있습니다.

이후 DenseFeatures를 통하여 모델에 주입

model = tf.keras.Sequential([
  feature_layer,
  layers.Dense(128, activation='relu'),
  layers.Dense(128, activation='relu'),
  layers.Dense(1, activation='sigmoid')
])

model.compile(optimizer='adam',
              loss='binary_crossentropy',
              metrics=['accuracy'])

model.fit(train_ds,
          validation_data=val_ds,
          epochs=5)
