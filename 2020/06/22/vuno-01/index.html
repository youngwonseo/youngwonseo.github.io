An Algorithm Based on Deep Learning for Predicting In-Hospital Cardiac Arrest

June 22, 2020 • Youngwon Seo

논문링크

Absract

  • 4개의 vital sign을 사용해 심정지와 사망에 대한 위험도를 예측
    • systolic blood pressure
    • heart rate
    • respiratory rate
    • body temperature
  • 2010.06 ~ 2017.07에 수집한 52,131명의 환자 데이터를 사용
    • 2010.06~2017.01 학습 데이터로 사용
    • 2017.02~2017.07 테스트 데이터로 사용
  • 순환신경망을 사용
    • 순환신경망 기반의 LSTM을 사용
  • 기존의 방법보다 높은 sensitivity와 낮은 false-alarm을 보임
    • sensitivity - event가 발생했다고 예측하는 것 중 실제 event가 발생한 비율
    • 결국 sensitivity가 높다는것은 낮은 false-alarm을 의미

Background

  • 80%의 심정지 환자는 악화증상을 심정지 8시간 전에 보임
  • 생존율이 20%이하를 보임(미국 기준)
  • 심정지를 예측하기 위해 TTS(track-and-trigger system)기반의 RRS(Rapid response system) 사용 됨
    • 각 vital sign 수치의 임계값(upper, under)을 기반으로 검사
    • 각 vital sign에 가중치를 사용해 계산후 합계를 기반으로 score를 계산
    • 정해진 가중치 값들을 이용해 계산한 결과를 MEWS(the modified early warning score)라고 함
  • 많은 이전 연구들은 sensitivity을 개선하는것에 초점을 둠
    • false-alarm에 대한 의료진의 시간낭비, 운영 비용증가
  • 본 연구에서는 DEWS(deep learning based early warning system)을 제안
    • 높은 sensitivity와 낮은 false-alarm의 성능
    • vital sign과 심정지의 관계를 deep learning으로 학습
