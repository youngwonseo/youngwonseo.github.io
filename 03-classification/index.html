분류

2019-10-01 / Youngwon Seo

본 글은 핸즈온 머신러닝을 요약한 글입니다.

3장의 주제는 분류(Classification)입니다. 다음과 같은 요소들에 대해 학습합니다.

  • 이진/다중 분류
  • 분류에 대한 성능 측정
    • 교차검증
    • 오차행렬
      • 정밀도(precision), 재현율(recall), F1 점수(F1 score)
      • 정밀도와 재현율의 트레이드 오프
    • ROC 곡선(receiver operating characteristic)
  • 에러분석
  • 다중 레이블 분류

분류를 학습하기위해 숫자 손글씨 데이터(MNIST)를 사용합니다. 머신러닝 또는 딥러닝 분야 학습에서 분류에 대한 입문을 위해 많이 사용되는 데이터셋 입니다. 사이킷런에 내장되어 있어 쉽게 받을 수 있습니다(참고로 Keras에도 내장 되어 있습니다).

from sklearn.datasets import fetch_openml
mnist = fetch_openml('mnist_784', version=1)

위에서 받은 mnist는 숫자 손글씨에 대한 이미지(data, 정확하게 설명하면 숫자 손글씨 이미지에 포함된 그레이 컬러셋의 픽셀정보)와 숫자 손글씨에 대한 숫자(target)로 구성되어 있습니다(그외 frame, feature_names, DESCR등의 정보가 포함되어 있습니다).

X, y = mnist['data'], mnist['target']
print('shape of X', X.shape)
print('shape of y', y.shape)

shape of X (70000, 784)
shape of y (70000,)

mnist의 data와 target은 numpy배열로 구성되어 있습니다. 먼저 X의 (70000, 784)은 70000x784 행렬을 의미 하는데 숫자 손글씨가 7만개로 구성되어 있고 각 손글씨는 784, 즉 28x28의 크기를 가진다는 것을 의미합니다. target은 각 손글씨에 대한 숫자(0,1,…,9)이기 때문에 70000x1, 즉 7만개를 가지는 것을 의미합니다.

손글씨 이미지에 해당하는 data는 784(28x28)차원(이미지의 픽셀)으로 구성되어 있습니다. 색상셋이 RGB가 아닌 그레이셋(흑백)이고 하나의 픽셀의 0~255(8bit)의 값을 가집니다. 다음은 7만개의 손글씨중 임의로 선택한 36001번째(0이 첫 번째 인덱스) 데이터를 시각화한 결과입니다.

%matplotlib inline
import matplotlib
import matplotlib.pyplot as plt

some_digit = X[36000]
some_digit_image = some_digit.reshape(28, 28)
plt.imshow(some_digit_image, cmap=matplotlib.cm.binary, interpolation="nearest")

숫자 9처럼 보이는데 target의 36001번째를 출력해 보겠습니다.

y[36000]

'9'

분류기를 학습하고 평가하기 위해서는 train set과 test set으로 구분이 필요합니다. 사이킷런의 mnist는 전체 7만개 데이터에서 앞쪽 6만개가 train set, 뒤 1만개가 test set을 의미합니다. test set에 대한 평가 이전에 검증을 위해 k-fold-validation을 사용하는데 데이터셋이 순차적(앞쪽 6천개가 0, 그다음 6천개가 1, .. 마지막 6천개가 9)으로 배치되어 있으면 모든 target에 대한 적절한 학습 및 검증이 수행되지 않습니다. 몇 가지 알고리즘은 이러한 순서에 민감해 같은 타겟을 가진 데이터가 연속해서 학습되면 성능이 나빠집니다. 이를 해결하기 위해 train set의 6만개의 데이터를 섞어 줍니다(사이킷런에서 제공되는 mnist는 이미 섞여있습니다).

import numpy as np

X_train, X_test, y_train, y_test = X[:60000], X[60000:], y[:60000], y[60000:]
shuffle_index = np.random.permutation(60000)
X_train, y_train = X_train[shuffle_index], y_train[shuffle_index]

다중분류

이번에는 여러개의 클래스를 구분하는 다중분류(multiclass classifier, multinomial classifier)로 , 다중 분류에는 다음과 같은 전략이 존재합니다.

  • 일대다(one-versus-all, one-versus-the-rest, OvA) : 각 클래스에 대한 점수를 통해 가장 높은 점수를 가진 클래스로 분류하는 전략
  • 일대일(one-versus-one, OvO) : 특정클래스와 나머지 클래스들과의 이진분류기를 훈련하는 전략, 클래스가 N개일때 (N X (N-1))/2 개의 분류기가 필요
