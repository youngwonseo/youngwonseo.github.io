시계열 데이터 분석 입문하기

2019-07-15 / Youngwon Seo

본 글은 시계열 데이터 분석에 앞서 시계열 데이터의 특성을 알아보고 시계열 데이터 분석을 위한 방법론들을 살펴보겠습니다.

1. 시계열 데이터란?

시계열(time series) 데이터란 일정한 시간 간격으로 배치된 데이터를 의미합니다. 즉 시간을 기준으로 생성(발생)된 데이터를 의미합니다. 이때 각 데이터들은 시간을 기준으로 앞, 뒤라는 순서가 존재하고 대부분의 경우 시간을 기준으로 의존성을 가지게 됩니다. 여기서 의존성이란 시간 t에 발생한 데이터는 시간 t-1의 데이터로 부터 의존적인 것이죠.

1.1 시계열의 구성 요소

그럼 먼저 시계열 데이터의 구성요소(특성)을 알아보겠습니다.

  • 추세 요인 : 지속적으로 시계열 데이터에 영향을 주는 장기 변동 요인을 의미(예: 기술의 변화, 소비형태의 변동, 인구 변동, 인플레이션 등)
  • 순환 요인 : 주기를 가지고 순환하는 요인으로써 중기 변동 요인을 의미(예: 경기 변동 곡선)
  • 계절 요인 : 계절을 주기로 발생하는 변동 요인으로써 1년을 주기로 나타나는 짧은 변동 요인을 의미
  • 불규칙 요인 : 어떤 규칙성이 없이 예측 불가능하게 우연적으로 발생하는 변동을 의미(추세변동, 순환변동, 계절변동 요인을 조정한 후에 나타나는 불규칙한 변동을 뜻함)

2. 시계열 분석을 위한 방법론

시계열 분석은 크게 계측과 해석으로 분류할 수 있습니다. 각 항목의 자세한 방법론은 추가되는 포스트에서 코드와 함께 설명하겠습니다. 여기서는 어떤 방법들이 있는지만 나열하겠습니다.

2.1 시계열 예측

  • 추세분석 (Trend Analysis)
  • 평활법 (Smoothing Method)
  • 분해법 (Decomposition Method)
  • 자기 회귀 누적 이동 평균 (Auto Regressive Integrated Moving Average : ARIMA)
  • 이분산 자귀회귀 모형 (Auto REgressive Conditional Heteroskedasticity)

2.2 시계열 해석

  • 스팩트럼 분석 (Spectral Analysis)
  • 개입분석 (Intervention Analysis)
  • 전이함수모형 (Transfer Function Model)
  • 자기상관오차를 갖는 회귀모형 (Auto Regressive Error Model)

3. 시계열 분석을 위한 절차

3.1 시계열 데이터 분해

3.1.1 추세 분해

Lowess/Loess 회귀(Regress)

  • 특정 범위에 적당한 다항 회귀선들을 구하여 병합
  • 다소 투박한(coarse) 추세선 : 회귀범위에 따라 달라짐

이동 평균 (Moving Average)

  • 특정 기간동안의 값의 평균의 변화 (예: 주가의 5일/10일 이동평균)
  • 부드러운 추세에 적용 가능 : 변화 주기를 알아야 함
  • 조밀한 (fine-grained) 추세선 Loess방법으로 1차 추세 분석 후, (반복적인) 이동평균 수행

3.1.2 주기 분해 및 변동 분해

  • 더하기(additive) 방식
  • 곱하기(multiplicative) 방식
