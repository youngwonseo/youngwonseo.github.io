추천시스템 간략소개(무비랜드 데이터셋을 활용한 추천시스템)

2019-07-09 / Youngwon Seo

사용자 기반 협업 필터링

사용자 기반 협업 필터링은 “당신과 비슷한 상품을 구매한 고객은 이런 상품도 구매했다”의 방법으로 사용자 구매목록을 기반으로 추천을 하는 방식입니다.

대표적 유사도로 피어슨 상관계수, 코사인 유사도, 자타드 계수가 존재합니다.

피어슨 상관계수(Pearson product-moment correlation coefficient)

코사인 유사도(Cosine similarity)

자카드 계수(Jaccard index, Jaccard similarity coefficient)

아이템 기반 협업 필터링

아이템 기반 협업 필터링은 사용자 기반 협업 필터링과 크게 다르 않게 주체가 사용자가 아닌 아이템인 방법입니다. 즉 “” 이 상품은 이 상품들과 유사하다”입니다.

모델 기반 협업 필터링

모델기반은 지도 또는 비지도 학습의 모델을 이용하여 군집화, 회귀 모델, 토픽 모델, 행렬 분해등에 따라 추천에 대한 항목을 예측하는 방법입니다.

내용 기반 협업 필터링

내용기반 필터링은 영화제목, 감독, 장르, 평점등을 이용하여 사용자 취향에 속한 영화를 추천하는 방식입니다. 즉 영화의 내용(속성)을 이용하는 것이죠.
